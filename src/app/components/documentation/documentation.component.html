Bonnes pratiques

Pour une boite claire

Traitez regulierement la boite de reception, deplacez les messages longs dans des dossiers et gardez les libelles pour les vues transversales.

Pour la securite

Activez la double authentification, surveillez les sessions actives et verifiez les expediteurs avant d ouvrir des pieces jointes inattendues.
